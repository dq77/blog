键盘使用说明书

发表于 2021-04-12 更新于 2022-05-10 分类于 说明书
Fn + Ins ：整体键盘背光颜色统一变化

1：常亮模式 2：游戏阶梯 3：呼吸模式 4：暗阴前行

Fn + Home ：触发一次键盘，灯光以按键为中心扩散或点亮

1：涟漪模式 2：踏雪无痕 3：单行涟漪 4：六心连锁

Fn + PgUp ：整个键盘背光快节奏的有规律的灯光变化

1：跑马灯 2：大鹏展翅 3：数码时代 4：推波助澜

Fn + Del ：整个键盘背光按规律动态流动

1：随波逐流 2：峰回路转 3：涌泉相拥 4：波光粼粼

Fn + End ：指点江山：自定义每个键的亮灭，默认心形图案

按Fn+Del进入该模式默认恢复上一次定义的效果，再次按Fn+End指示灯闪烁表示进入编辑模式，触发按键可切换LED的亮灭，定义完成后再次按Fn+End保存

Fn + PgDn ：整个键盘背光为多彩动态变化

1：星光闪烁 2：微波粼粼 3：流星赶月 4：彩旗飞扬

Fn + 方向键上/下 ：调节背光亮度
Fn + 退格左边两个加减号 ：调节流彩速度
Fn + 方向键左/右 ：调节流光方向
Fn + Esc ：长按恢复出厂设置
Fn + Pause ：暂停当前动态的背光模式
