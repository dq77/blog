Typora安装方式及快捷键整理

发表于 2021-06-09 更新于 2024-04-23 分类于 Typora

Typora安装

  1. 下载typora-0.11.18后直接安装，双击运行安装程序，最后会提示软件版本过期，无视掉。

    安装包在网盘里有，或者谷歌搜索。

  2. 按 Windows + R 打开运行窗口，输入regedit，点确定，打开注册表。

  3. 依次展开：计算机\HKEY_CURRENT_USER\Software\Typora，然后在Typora上右键，点权限，选中Administrtors，把权限全部设置为拒绝。
