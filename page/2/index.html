Typora安装

  1. 下载typora-0.11.18后直接安装，双击运行安装程序，最后会提示软件版本过期，无视掉。

  2. 按 Windows + R 打开运行窗口，输入regedit，点确定，打开注册表。

  3. 依次展开：计算机\HKEY_CURRENT_USER\Software\Typora，然后在Typora上右键，点权限，选中Administrtors，把权限全部设置为拒绝。

  4. 打开Typora软件，左上角点击：文件 → 偏好设置 → 自动检查更新。取消前面的勾选。

功能 亮度- 速度- 速度+ 锁WIN 灯条模式 灯条颜色

功能 展示电量 侧灯模式 侧灯颜色 侧灯亮度 侧灯速度
windows下F键快捷功能

功能 上一曲 播放/暂停 下一曲 静音 音量- 音量+

Fn + Ins ：15种RGB和关闭共16种灯效切换

随波逐流(默认)、如影随形、正弦光波、旋转风车、七彩瀑布、花开富贵、游戏模式、关闭、常亮模式、呼吸模式、梦幻彩虹、一触即发、雨中漫步、按键涟漪、繁星点点、川流不息。

Fn + Esc 长按3秒，恢复默认。

Fn + 左Win锁定/打开左WIN窗口键(锁时灯常亮)。

Fn + B切换到电量模式，充电时流光，充满常亮绿色，5个灯一个代表20%电量。

驱动教程及下载地址在这里，可音乐律动、颜色及按键自定义、宏定义。

2.4G对码

开关拨至2.4G模式， 键青色灯慢闪，然后长按3秒Fn + 青色灯快闪进入对码状态。此时插入接收器完成对码， ~键青色灯亮2s，2.4G连接成功。

蓝牙连接

设备名AULA-F87。

开关拨至蓝牙模式，可连接三个设备。Fn + 1/2/3通道广播时蓝灯快闪，连上亮2S。
