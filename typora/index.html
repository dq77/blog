Typora安装方式及快捷键整理

发表于 2021-06-09 更新于 2024-04-23 分类于 Typora

Typora安装

  1. 下载typora-0.11.18后直接安装，双击运行安装程序，最后会提示软件版本过期，无视掉。

    安装包在网盘里有，或者谷歌搜索。

  2. 按 Windows + R 打开运行窗口，输入regedit，点确定，打开注册表。

  3. 依次展开：计算机\HKEY_CURRENT_USER\Software\Typora，然后在Typora上右键，点权限，选中Administrtors，把权限全部设置为拒绝。

  1. 打开Typora软件（默认安装在C:\Program Files\Typora），左上角点击：文件 → 偏好设置 → 自动检查更新。取消前面的勾选。

MarkDown语法

各级标题 Ctrl + 1

斜体文本 和 粗体文本 以及 ***粗斜体文本 *** Ctrl + I Ctrl + B

  • 无序列表 Ctrl + Shift + ]
  1. 有序列表 Ctrl + Shift + [

区块引用 Ctrl + Shift + Q

函数块 使用 Ctrl + Shift + M

代码块的使用 Ctrl + Shift + K

链接 Ctrl + K

图片 Ctrl + Shift + I

表格 Ctrl + T

转义：\

正常显示俩星号 **
